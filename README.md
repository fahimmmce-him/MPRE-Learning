# MPRE Learning System

React + Vite web app for MPRE course learning management with Firebase Firestore realtime sync.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Upload this project to GitHub, then import the repository in Vercel.

## Firebase

The Firebase config is already placed in `src/App.jsx`. Make sure Cloud Firestore is enabled in Firebase Console.
